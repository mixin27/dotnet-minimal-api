## Installation

```bash
# clone the repository
$ git clone https://github.com/mixin27/dotnet-minimal-api.git

# change directory
$ cd dotnet-minimal-api/

# restore project
$ dotnet restore

# change directory to the actual api folder
$ cd src/DotNetDemo.Api/

# run the app
$ dotnet run

```
